#  YouTube AI Summarizer

An AI-powered YouTube transcript summarizer built using Python, Streamlit, Groq LLM, and YouTube Transcript API.

The application extracts transcripts from YouTube videos and generates concise AI-based summaries with important key points.

---

## Features

- Extract YouTube video transcripts automatically
- Generate AI-powered summaries using Groq LLM
- Supports English and Hindi subtitles
- Displays YouTube video thumbnail
- Clean Streamlit UI
- Handles long transcript token limits
- Fast inference using Groq API

---

##  Tech Stack

- Python
- Streamlit
- Groq API
- YouTube Transcript API
